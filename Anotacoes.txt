LEMBRETES
💡 Regra prática profissional

** Use srcSet quando:
- A imagem passa de ~100–150KB
- Ou aparece muito no site
- Ou precisa máxima qualidade visual
- Caso contrário → uma imagem otimizada já resolve.

- EXEMPLO DE USO srcSet:<img
  src="/logo-96.png"
  srcSet="/logo-64.png 64w, /logo-96.png 96w, /logo-128.png 128w"
  sizes="(max-width: 768px) 64px, 96px"
  alt="Logo"
/>

4️⃣ Como rodar o projeto
### 1. Clonar o repositório

git clone https://github.com/Elaine-Tavares/portfolio_elaine_tavares.git

### 2. Instalar dependências do frontend

npm install

### 3. Iniciar o frontend

npm run dev

O projeto estará disponível em:

http://localhost:5173

### 4. Configurar o backend

O backend foi desenvolvido em PHP com banco de dados MySQL.

É necessário:

- Servidor local (XAMPP, WAMP ou similar)
- MySQL configurado

Importe o banco de dados e configure o arquivo de conexão com as credenciais locais.
